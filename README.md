# QB-Assistant

## Features

### Excel Export Framework
Professional Excel workbook generation using openpyxl library. Supports multiple sheet types with consistent formatting:

- **Executive Summary**: High-level financial metrics in key-value format
- **KPI Dashboard**: Financial metrics (current ratio, burn rate, cash runway) across periods
- **Budget vs Actual**: Variance analysis with absolute and percentage differences
- **Cash Flow Forecast**: Operating, investing, and financing activities with confidence intervals
- **P&L Forecast**: Income and expense hierarchies with projected, lower bound, and upper bound values

All sheets include professional formatting (currency, percentages, borders, auto-sized columns) and are compatible with Excel on macOS.
